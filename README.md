## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK (^3.9.2)
- Dart SDK
- Android Studio / VS Code
- AWS Amplify CLI (for backend)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Manushanth1/PhoneWork.git
   cd PhoneWork
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Configure Amplify** (if using backend)
   ```bash
   amplify init
   amplify push
   ```

4. **Run the app**
   ```bash
   flutter run
   ```
